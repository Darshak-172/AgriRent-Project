# AgriRent-Project
<h3>AgriRent - Equipment Rental and Farm Supplies Selling System</h3> <br><br>
<b>AgriRent</b> is a comprehensive web-based system designed to connect farmers and equipment owners within a district, enabling farmers to rent agricultural machinery and purchase essential farm products online.<br><br>
The platform facilitates easy registration and login for different user types (farmer, equipment owner, admin), offers searchable listings for equipment and products, supports booking and ordering with real-time availability checks, and integrates secure payment processing through Razor pay payment gateway supporting multiple payment methods including UPI, credit/debit cards, net banking, and digital wallets.<br><br>
<b>AgriRent</b> includes features for reviews, complaints, admin approvals for quality control, and provides scalable, secure, and mobile-friendly access to make modern farming resources more affordable and accessible to local agriculturalÂ communities.
